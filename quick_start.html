專案KDD流程

KDD（Knowledge Discovery in Database）是從資料中辨別有效的、新穎的、潛在有用的、最終可理解的模式的過程； 而資料探勘是KDD通過特定的演算法在可接受的計算效率限制內生成特定模式的一個步驟。
本系統的KDD流程分為四大部份：Data Pre-process, Feature Selection, Model Training, Model Prediction

0.Data Input
