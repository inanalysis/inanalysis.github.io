How Does It Work?

InAnalysis Work Flow
Project Management

能夠察看所有專案相關資訊, 新增專案, 修改專案, 刪除專案。新增專案後點選KDD，進入專案KDD流程
Can view all project related information, add projects, modify projects, and delete projects. After adding the project, click KDD to enter the project KDD process.

KDD Process

Data Pre-process, Feature Selection, Model Training, Model Prediction。訓練出Model後可以在Model Management建置API
After training the Model, you can build the API in Model Management.

API Management

能夠察看使用者建立的API, 修改API, 刪除API。透過API服務，輕鬆將模型佈署至任何應用。
Ability to view user-created APIs, modify APIs, and remove APIs. Easily deploy models to any application through API services.
